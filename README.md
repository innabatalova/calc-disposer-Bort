# calc-disposer-Bort
Калькулятор выгоды использования промышленных измельчителей для ресторанов.

# Technologies used
- React
- React Hooks
- Css-modules
- MUI-Material

# Install and start
"$" npm run install - установка зависимостей

"$" npm run build - сборка проекта

"$" npm run start - запуск проекта
